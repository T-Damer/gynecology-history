# Gynecology history

Frontend app for creating patient cards, storing shared passport data, adding an unlimited number of gynecology visits, and exporting the result to Excel.

## What it does

1. Creates a patient card with shared passport fields.
2. Stores any number of visits under the same patient.
3. Persists data in local storage.
4. Exports one Excel row per visit, repeating the passport data.
5. Imports previously exported Excel files back into the app.

## Local launch

1. Install Node.js LTS.
2. Install dependencies with `yarn`.
3. Run `yarn start`.
4. Open the local Vite URL from the terminal.
